ICAP Time Tracking App.

## Userscreens
Please refer to USER MANUAL DOCUMENT (Word)

- Create an account, first user is assigned SUPER USER rights, and does not have a Supervisor
- Login using your account
- Create a timesheet by adding a new timesheet using the "New Timesheet" button

## Requirements

For building and running the application you need:

- [JDK 1.8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
- [Maven 3](https://maven.apache.org)
- [MySQL 5](https://www.mysql)

## Running the application locally

```shell
mvn spring-boot:run
```
